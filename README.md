# Smart Home AIoT Monitoring System
LD7182 - AI for IoT | Northumbria University

## Description
ESP32-S3 based smart home monitoring system with 5 sensors,
edge AI classification and ThingSpeak cloud dashboard.
Monitors residential safety and energy waste in real time.

## Hardware
- ESP32-S3 microcontroller
- DHT22 temperature and humidity sensor
- MQ2 gas and smoke sensor
- PIR motion sensor
- LDR photoresistor
- HC-SR04 ultrasonic sensor
- SSD1306 OLED display
- Buzzer and LED

## Tools Used
- Wokwi simulation platform
- ThingSpeak cloud dashboard
- Google Colab machine learning
- Edge Impulse TinyML

## Files
- main.cpp — ESP32-S3 firmware
- diagram.json — Wokwi circuit
- libraries.txt — Required libraries
- LD7182_Smart_Home_AIoT.ipynb — Google Colab ML notebook
